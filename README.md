# 猜动物游戏

一个简单有趣的猜动物小游戏，支持渐进式提示功能。

## 游戏特色

- **20种常见动物**：包含狗、猫、鸟、鱼等多种动物
- **渐进式提示**：三条提示按照从模糊到精确的顺序排列
  - 第一条：最模糊，提供广泛类别信息
  - 第二条：中等模糊，提供更具体特征
  - 第三条：最精确，提供非常具体的行为或特征
- **计分系统**：根据使用提示的次数获得不同分数
  - 无提示：15分（基础10分+额外5分）
  - 1次提示：13分（基础10分+额外3分）
  - 2次提示：11分（基础10分+额外1分）
  - 3次提示：10分（仅基础分）
- **响应式设计**：适配不同屏幕尺寸

## 如何使用

1. 双击打开 `index.html` 文件
2. 在输入框中输入你猜测的动物名称
3. 点击"猜一下"按钮检查答案
4. 点击"获取提示"按钮获得帮助（最多3次，从模糊到精确）
5. 点击"重新开始"按钮开始新游戏
